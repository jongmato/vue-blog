# Vue & Firebase를 이용한 블로그 사이트 제작

## front
- 프레임워크 : VueJS(버전2)
- UI 디자인 : Vuetify
- 상태관리 : vuex

## Firebase
- Firebase Functions
- Firestore : 게시판에 사용되는 DB
- Realtime Database : navbar title, footer 등 layout수정용 DB
- Hosting
- Authentication : 이메일인증, 구글인증 사용
